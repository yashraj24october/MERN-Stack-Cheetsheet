First, we create a server and then in server callback method, we create our header using writeHead()
And then return the data using write() and then end the api

As:

http.createServer((req,res) =>{
res.createHead(200//that is status, {'Content-Type':'Application\json'});

res.write(JSON.stringify({name:yash, profession: software engineer}));

res.end();

}).listen(1000);
