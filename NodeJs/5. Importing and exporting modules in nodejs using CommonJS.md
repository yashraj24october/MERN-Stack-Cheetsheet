In nodejs, modules are the files that we used to make resusable data and function that can be easily accessible from any file.

Just we need to export the data and function and then by importing the file, we can access all its exported data and function.
There are two way in NodeJS for export and import:
1. Using CommonJS way
	2. Using ESM way


Using CommonJS
	----------------------------

	
**Exporting data
**
To export data and functions, we can use module.exports in file.
-------------------------------------------
module.exports = {
data_key: value,
data_key: value,
data_key: value
}
//module.exports is used to export all the data 
// from a file


Importing module to access data 
----------------------------------------------------------
to import a module, we use require()

let module = require(./exported_file_name.js);

//require() is used to import any module/file

to access any exported data, we can use 
module.dataName
